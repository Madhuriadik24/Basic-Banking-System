# Basic-Banking-System
This is a simple dynamic website project for  The Spark Foundation. A web application used to transfer money between two users.
# Technologies Used:
# Front-End:
* HTML
* CSS
* Bootstrap
# Back-End:
* PHP
* MYSQL
